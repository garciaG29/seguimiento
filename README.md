# Proyecto Laravel - Ejercicios MVC

Este repositorio contiene ejercicios básicos del patrón *Modelo - Vista - Controlador (MVC)* realizados con Laravel como seguimiento del aprendizaje.

---

## Contenido del repositorio
- Código fuente de Laravel
- Ejemplos prácticos del patrón MVC
- Controladores, modelos y vistas creadas durante el aprendizaje

---

## Objetivo del proyecto

El objetivo es aprender los conceptos fundamentales de Laravel utilizando rutas, controladores, modelos, migraciones y vistas.

---

# Tecnologías utilizadas

- PHP (Laravel)
- Blade Templates
- MySQL (o SQLite)
- Composer
- Git / GitHub
